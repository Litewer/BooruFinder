# Booru Finder 🚀

Удобный клиент для поиска и просмотра контента из `rule34.xxx` и `gelbooru.com`.

## Что умеет ✨
- Поиск по тегам и blacklist-тегам
- Сортировка (`Новые` / `Популярные`) и фильтр по `score`
- Просмотр картинок и видео во встроенном плеере
- Избранное для видео
- Скачивание файлов в папку загрузок
- Отдельные API-ключи для каждого сайта

## Важно по ключам 🔐
- В проекте **нет ваших личных ключей**.
- Пользователь вводит свои `user_id` и `api_key` сам в интерфейсе.
- Ничего не публикуйте в GitHub с заполненными ключами.

## Как скачать готовую версию 📥
После публикации релиза на GitHub:
1. Открой страницу `Releases`.
2. Скачай:
   - `BooruFinder.exe` (Windows)
   - `BooruFinder-android-debug.apk` (Android)

## Как установить 🛠️

### Windows (EXE)
1. Скачай `BooruFinder.exe`.
2. Запусти файл двойным кликом.
3. При первом запуске заполни свои API-данные в блоке ключей.

### Android (APK)
1. Скачай `BooruFinder-android-debug.apk`.
2. Разреши установку из неизвестных источников.
3. Установи APK и запусти приложение.
4. Введи свои API-данные в приложении.

## Как пользоваться 📚
1. Поставь галочку `Мне есть 18 лет`.
2. Выбери источник (`Rule34`, `Gelbooru` или оба).
3. Введи теги в поле `Теги`.
4. Нажми `Поиск`.
5. Открой карточку для просмотра.
6. Для сохранения видео в избранное нажми `Лайк`.
7. Для скачивания нажми `Скачать`.

## Где взять API ключи 🔑

### Rule34
1. Зайди в настройки аккаунта.
2. Найди блок `API Access Credentials`.
3. Скопируй `user_id` и `api_key`.

### Gelbooru
1. Зайди в профиль/настройки API.
2. Скопируй `user_id` и `api_key`.

## Сборка из исходников (для разработчика) 👨‍💻

### Windows EXE
```powershell
python -m pip install -r requirements.txt
.\build_exe.bat
```
Готовый файл: `dist\BooruFinder.exe`

### Android APK
```powershell
.\build_android_apk.bat
```
Готовый файл: `dist\BooruFinder-android-debug.apk`

## Безопасность и приватность 🧩
- В приложении используются только ключи пользователя.
- Ключи можно очистить кнопкой `Очистить secure-хранилище`.
- Если нужен прокси, укажи его в поле `Proxy URL`.
